Each stroke is a centerline path, so when you convert the OTF to SVG the pen plotter draws each letter as one line instead of tracing both edges. Preview above shows the dimensional version; the line font may not render correctly in browsers.

How do I install my font?

Mac: double-click the downloaded .otf → click Install Font. It appears in every app's font menu.

Windows: right-click the .otf → Install (or Install for all users).

iPhone / iPad: install a font manager like iFont from the App Store, open the .otf with it, then allow the profile in Settings.

In documents: after installing, pick the font by the name you gave it. Alternates cycle automatically in apps that support contextual alternates (Pages, Word, browsers).
